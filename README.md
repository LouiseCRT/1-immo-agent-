# Versailles Immo Alert

Système de veille immobilière automatisée pour Versailles (78000).

Objectif :
- récupérer les annonces immobilières
- calculer le prix au m²
- comparer les biens au marché local
- classer les meilleures opportunités
- envoyer un rapport quotidien par email

Le projet est prévu pour fonctionner automatiquement avec GitHub Actions.
